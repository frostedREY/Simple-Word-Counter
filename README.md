# Word Counter 📝

A simple yet powerful **Word Counter Web App** built with **HTML, CSS, and JavaScript**.  
It allows users to quickly analyze text, providing both **basic statistics** (words, characters, spaces, dots) and **advanced insights** (sentence count, paragraph count, reading time, average word length, longest/shortest word).

---

## 🚀 Features

### 🔹 Basic Counters
- **Word Count** – total number of words
- **Character Count** – total characters including spaces
- **Space Count** – number of whitespace characters
- **Dot Count** – number of periods (`.`)

### 🔹 Advanced Features (toggleable)
- **Sentence Count** – number of sentences
- **Paragraph Count** – number of paragraphs
- **Estimated Reading Time** – based on ~200 words per minute
- **Average Word Length** – mean length of words
- **Longest Word** – longest word in the text
- **Shortest Word** – shortest word in the text
